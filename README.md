# Quantum Mania
This repo contains some resources and links for my "Quantum Mania" talk, and will receive occasional updates as I find cool new things to share. These are just some personal favorites and things I specifically call out in my talk. For way more content, check out [Awesome Quantum Computing](https://github.com/desireevl/awesome-quantum-computing)!
